# MaskedEditText

It is project that extend standard UITextView and UITextField. The main thing is adding a type parameter. 
And it is the only thing you need to set to get some input settings done.

## Features

### UITextView and UITextField
* Automatic setting keyboardType
* Automatic setting autocapitalizationType
* Additional done button on keyboard types without return button

### UITextView
* MultiLines compatibility
* Added placeholder

Written in Swift
